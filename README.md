# DNA Sequencing Data Analysis

This repository contains workflows, scripts, and analysis pipelines for processing and analyzing DNA sequencing (DNA-seq/NGS) datasets. The project includes:

- Quality control (FastQC, MultiQC)
- Read trimming and preprocessing
- Genome alignment
- BAM/SAM processing
- Variant calling (SNPs/INDELs)
- Functional annotation
- Statistical analysis and visualization

Tools Used:
- FastQC
- BWA
- SAMtools
- GATK
- BEDTools
- Python
- R

The workflows are designed for reproducible and scalable genomic data analysis.
